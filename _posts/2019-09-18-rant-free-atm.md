# Strangely rant free
I'm strangely rant free today. I was reading David Graeber's _Bullshit Jobs_ this morning, but it did not incense me. I thoroughly enjoy it, but at the same time it makes me think about the bureaucratic system that I live and work in.

Today I'm scheduling future meetings and getting stuff going. I don't think that I have a bullshit job today. But I'm also doing bioinformatic stuff today. I'm working on getting ExomeDepth to work. This is an R package for computing and annotating copy number variants (CNVs) from whole exome sequences (WES). I've jumped in deep, and now I'm happy to say that I've gotten it to work. We're able to see the CNVs from the sequences that we're working with. We're also able to have annotations from several databases. Right now we're working with the names of the genes that the CNVs touch and the CNV ISCA values. I'm not quite sure what ISCA is, but it returns the pathogenicity value.

This is relatively easy because the databases that we are working with on using the same reference genome as the WESs. I'm going to try to get my scripts to work so that I can use older version of databases and the data that is associated with them. I want to be able to "liftover" information that is old to the newer reference information. Theoretically that shouldn't be that hard, but I'm running into issues with versioning. I'm also running into issues with my servers. Basically, my experience with linux and *nix systems allows for me to be fairly agile, but I don't have that much genetics/bioinformatics experience. Or I didn't at the beginning of this. I think that I do now. I am working on pairwise alignment of reference genomes depending on the databases. I've created another project that is just as big, if not bigger than the original project. I've always thought that Matryoshka dolls were supposed to get smaller, but that is not the experience I'm having.
