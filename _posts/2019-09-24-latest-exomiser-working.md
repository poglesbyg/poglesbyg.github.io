# It's working!
Yes, I do feel like I'm trying to get a pod-racer working. Do I have a snarky to-be-golden robot? Nope. On the other hand, I feel like I could name my expressions R2D2, because they very often have to be bleeped. Java just makes me unhappy. Might it be the lack of documentation? That could be very, very possible. I remember a graph about the amount of profanity used in the comments of a specific language in Github. It has been a long time since I saw that graph. I'm pretty sure that I remember that C and C++ had the most profanity, but I think that I also remember that at this point in time C and C++ were the most used languages. It would make sense that the most committed/used langugaes would have the most comments and the most profanity. I learn to program Java, so that is my profane language. I don't find myself cursing Python or Julia. I'm slightly older now and I don't use profanity as much in my daily language. I think that this is an age thing. I'm regressing to how I used to speak when writing in a different language. It would be interesting to see how my state of mind would change if I went back to speaking Norwegian. I would hope that I'd be able to maintain my current state of mind, but I would also guess that it would change to some degree too. How do I collect data on this question?
